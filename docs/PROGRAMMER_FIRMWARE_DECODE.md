# Расшифровка прошивки автора: подтвержденное

Файл намеренно очищен от DBC-кандидатов и старых догадок. Здесь только то, что
можно использовать как рабочую базу.

## Базовая архитектура

Штатная прошивка автора работает в двух практических состояниях:

- `canbox` - основная работа;
- `update` - штатная USB-загрузка обновления.

Модель работы canbox:

```text
CAN автомобиля + физические входы + UART/USB команды от магнитолы/APK
    -> логика прошивки автора
    -> CAN автомобиля / UART магнитолы / reverse output
```

Значит APK не обязан слать навигацию, FM и музыку как raw CAN. Он может слать
команды в формате прошивки, а прошивка уже сама строит CAN-кадры.

## Подтвержденные чтения C-CAN

| Функция | CAN ID | Байты | Формула / бит |
|---|---:|---|---|
| Левая передняя дверь | `0x541` | `DATA[1] bit0` | `1 = открыто` |
| Правая передняя дверь | `0x541` | `DATA[4] bit3` | `1 = открыто` |
| Багажник | `0x541` | `DATA[1] bit4` | `1 = открыто` |
| Капот | `0x541` | `DATA[2] bit1` | `1 = открыто` |
| Левая задняя дверь | `0x553` | `DATA[3] bit0` | `1 = открыто` |
| Правая задняя дверь | `0x553` | `DATA[2] bit7` | `1 = открыто` |
| Задний ход | `0x169` | `DATA[0] & 0x0F` | `0x07 = R включен` |
| Скорость | `0x316` | `DATA[6]` | используется прошивкой как скорость |
| Температура двигателя | `0x329` | `DATA[1]` | `(raw - 0x40) * 0.75` |
| Наружная температура | `0x044` | `DATA[3]` | `(raw - 0x52) / 2` |

## Подтвержденные media/source входы

Семья Raise/HU:

```text
FD .. 09 <source> ...
```

| Источник | Кадр |
|---|---|
| FM | `FD 08 09 02 00 65 46 00 BE` |
| USB music | `FD 0A 09 16 00 00 00 00 02 00 2B` |
| Bluetooth music | `FD 06 09 11 00 00 20` |
| Navigation source | `FD 06 09 06 00 00 15` |

Подтвержденные USB/API команды прошивки:

| Команда | M-CAN ID | Роль |
|---:|---:|---|
| `0x20` | `0x4E8` | FM/station text |
| `0x21` | `0x4F6` | media/source text |
| `0x22` | `0x490` | USB/track text |

## Что больше не считать подтвержденным

Следующее убрано из рабочей базы до нового чистого доказательства:

- люк как штатная функция прошивки автора;
- подогрев руля;
- TPMS;
- RCTA/RCCW/BCW exact bits;
- ADAS/SVM;
- `0x114/0x197` как обязательная схема медиа/навигации;
- CarPlay/Android Auto raw CAN IDs;
- clean-C прошивка как замена авторской.
