# CAN-C / M-CAN data

Здесь только то, что относится к CAN-C/M-CAN работе машины.

Файлы:

- `confirmed_can_signals.csv` - короткая подтвержденная база для приложения.
- `LIVE_CAN_C_M_CAN.md` - короткая live-карта: что сидит в CAN-C и M-CAN,
  включая напряжение, parking raw и RCTA-кандидаты.
- `learned/learned_assignments.jsonl` - сохраненные результаты обучения из dashboard.

Старые DBC-only и media/nav candidate-таблицы удалены из рабочей документации,
потому что они мешали проверке. Их можно восстановить из истории git, если
появится конкретный тест.
