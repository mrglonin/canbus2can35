# Simple Soft / 2E UART

Отдельная ветка данных для Simple Soft / XP / `2E`-протокола.

Файл:

- `simple_soft_uart_full_worklist.csv` - список кадров для проверки, если на
  магнитоле будет выбран Simple Soft/Kia/Hyundai профиль вместо Raise/RZC.

Это не основной профиль текущей машины. Использовать только для отдельного
теста с переключенным профилем canbox в TEYES.
